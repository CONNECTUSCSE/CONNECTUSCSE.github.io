Features of ConnectUs

CONNECT FEED

Connect feed is a combined live feeding platform on News, Updates , Students, Clubs, Departments and University.

TALENT KEN

Talent Ken is a platform for students to showcase their talents.

TRENDING QUERIES

Trending Queries is a platform where the students can post questions with images.Their questions can be answered in the comment section.The queries with most likes will be displayed first.

SHERLOCK BOX

Sherlock Box is a forum for students to post and find their lost belongings.

ALUMNI CONNECT

Alumni Connect is a social network based on private alumni networks focused on helping an alum / student get trusted help from his/her alumni network. Help ranging from getting referral for jobs, advice for careers, recommendations for services.

CONNECT POLL

Connect poll is the platform where the professors can post questions in the form of a poll.

LECTURE BANK

Lecture bank is a platform for the students to access the lecture related files posted by the professors.

EVENT REGISTRATION

Event Registration can be used by the various clubs in the college to post events and the students can directly register through CONNECT US.

FEEDBACK

This is the place where the users can post their feeback about CONNECT US. This will help us to improve our application.

CLASSROOM

Classroom is a messaging service for the students, professors and the various club members. Here, the users can also share images.

PLATFORM USED

Our team has built ConnectUs using React Native and Firebase

ABOUT REACT NATIVE AND FIREBASE

React Native is an open-source mobile application framework created by Facebook.It is used to develop applications for Android, iOS, Web and UWP by enabling developers to use React along with native platform capabilities.

Firebase is a Backend-as-a-Service — BaaS — that started as a YC11 startup and grew up into a next-generation app-development platform on Google Cloud Platform.Firebase frees developers to focus crafting fantastic user experiences. You don’t need to manage servers. You don’t need to write APIs. Firebase is your server, your API and your datastore, all written so generically that you can modify it to suit most needs.
